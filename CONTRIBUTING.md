# Contributing to OpenAnimator NG

Thank you for wanting to help build the open raster animation workstation

## How to contribute
1. Fork the repository
2. Create a branch for your feature or fix
3. Test on Windows, macOS, or Linux
4. Follow the existing C style for core changes
5. Add Python plugins in the plugins/ folder
6. Submit a pull request with clear description

We welcome contributions on:
- Raster and compositing improvements
- Node system enhancements
- Python plugin examples
- Cross-platform build fixes
- Documentation and templates

Open issues and discussions are welcome on any platform.
