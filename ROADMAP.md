# OpenAnimator NG - Roadmap

## Phase 0 - Foundation
- Repository created and branded
- Builds and runs on Windows, macOS, and Linux

## Phase 1 - Core Modernization
- True-color canvas support
- Unlimited undo/redo with delta storage
- Basic layers and masks
- Onion skin and animation curves
- Cross-platform build system finalized

## Phase 2 - Node System
- Schematic view for piping cells, colors, effects, transforms
- Non-destructive compositing

## Phase 3 - Python Integration
- Python plugin loader
- First plugins: procedural effects and export tools

## Phase 4 - Production Pipeline
- Scenes, sequences, and object library
- Animation library and presets
- Asset browser with sounds, images, textures

## Phase 5 - Ecosystem
- Built-in and community plugins
- Marketplace for objects, effects, and presets
- Export pipelines for video, sprite sheets, and large formats

## Phase 6 - Polish
- Real-time playback optimizations
- Community templates for TV intros, cartoons, games, broadcast graphics
- Official releases for Windows, macOS, and Linux
